An aspiring Software Engineer

Hi there! My name is Stella and I’m currently attending Software Engineering at McMaster University.

As an aspiring software engineer, I chose Engineering as I enjoy coming up with innovative solutions to problems that come up in everyday life. I also find that being able come up and create solutions and products is a great way to challenge myself and learn new skills.

I like to keep myself busy with hands on activities such as making food, building something (including 3D puzzles and putting together furniture), programming codes that provide a variety of functions, arts and craft, and solving puzzles.

Projects

  • Community Challenge

    The goal of this project was to create a solution that would facilitate the daily life of our client, who has spina bifida and gets around using a wheelchair. The final solution was a console cover that could protect the console from water damage during harsh weather conditions such as rain, snow, or hail while allowing the client to comfortably maneuver her wheelchair.

  • Healthcare Challenge

    The healthcare challenge that we were supposed to solve was, using a robotic arm known as the Q-arm, to pick up, transfer, and drop off a surgical tool container into an autoclave to sterilization

  • Connect Four Game

    A game of Connect Four that is adapted so that it immerses elements of Pokémon. This game is a 2 player game and is user friendly. This game includes the help page, score page, game page and exit page, which are all accessible, and can easily be navigated by users. As well, there is an easter egg that can be found when exiting the game!
